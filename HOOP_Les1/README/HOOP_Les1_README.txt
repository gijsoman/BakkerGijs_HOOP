Er zijn een aantal objecten die volgens mij een rol spelen:
- We hebben object nodig die de ballen spawned op een random locatie.
- we hebben een object nodig dat de ballen doet laten pulseren.
- We hebben een object nodig dat de ballen laat bewegen met een bepaalde snelheid.

Er zijn ook een aantal attributen die nodig zijn:
- Voor het spawnen van de ballen:
	- Een x en een y locatie
	- De breedte en hoogte van het scherm zodat je ze binnen het scherm kan laten spawnen
	  (hier hoef je geen aparte variabelen voor te maken deze zijn rechtstreeks aan te spreken)
	- Een radius met hoe groot de bal moet zijn
- Voor het laten pulseren van de ballen:
	- Hier heb je ook de radius voor nodig.
	- Ook heb je hier een waarde nodig met hoeveel je de ballen laat pulseren.(pulseringswaarde)
- Voor het laten bewegen van de ballen:
	- Hier voor heb je een waarde nodig die ze over de x-as beweegd(x-speed)
	- Ook heb je hier een waarde voor nodig die ze over de y-as beweegd(y-speed)
